Video Streaming Test for Redis
===============

Video Streaming with websocket. 

Installation
------------

```
$ sudo apt-get install python-opencv
$ sudo apt-get install redis-server


$ pip install coils
$ pip install numpy
$ pip install redis
$ pip install tornado
```

Usage
-----
```
$ python producer.py
$ python consumer.py
```

   
Go to http://localhost:9000 to view the webcam.
